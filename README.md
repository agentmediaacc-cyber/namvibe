# NamVibe

NamVibe is a global dating, chat, wallet, video call, and live streaming platform.

## Main folders

- api_routes: all page/API routes
- middleware: login/session/security checks
- services: business logic and Supabase calls
- models: data structure helpers
- templates: HTML pages
- static: CSS, JS, images
- config: app settings
- utils: Supabase client and helper functions
